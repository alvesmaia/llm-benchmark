# Fase 1 — Construção

Implemente o projeto descrito abaixo **completamente**, do zero, no diretório de trabalho atual.
Trabalhe de forma autônoma: crie todos os arquivos, instale o que precisar, e não pare para pedir
confirmação. Ao terminar, garanta que o projeto está completo (todos os entregáveis obrigatórios presentes).

O diretório já é um repositório git inicializado (não rode `git init`). **Não** faça commit nesta fase —
isso será pedido na Fase 3.

A base de dados de teste está em `DNE_PATH` (já exportada no ambiente). Use-a para validar o ETL.
